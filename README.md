# java_itellij_IDEA

在 Windows 系統下安裝 itellij IDEA 

進入官網(https://www.jetbrains.com/idea/)下載 itellij IDEA 的可執行文件

有分Ultimate(旗艦版) 和 Community(社區版)

旗艦版是收費的，而社區版是免費的，旗艦版的筆社區版的功能更全面，安裝的項目基本上都是下一步就可以了。
